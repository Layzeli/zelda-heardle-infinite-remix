# Credits

This project is a personal, non-commercial modification inspired by two existing projects:

* **Mario Heardle Infinite** by NinjiGalaxy  
Original codebase used as the foundation for this version. Mario Heardle was built on Etrian Odyssey Heardle Infinite by Beigent0. That project originates from a Heardle template that can be traced back to WJSN Heardle (created by jeroldcamacho), which in turn was a fan-made clone of the **original Heardle**, later shut down after being acquired by Spotify.

* **Zelda Heardle** by DarkDecay
Song links and soundtrack references were taken from the original Zelda Heardle.

Additional contributions:

* Volume slider and scroll bar improvements by morgvanny.
* Personal modifications, UI adjustments, metadata creation and game logic fixes by @layzeli.

This project is not affiliated with Nintendo or any of the original authors.  
It is distributed solely for educational and entertainment purposes.
