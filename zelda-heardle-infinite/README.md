# Zelda Heardle Infinite

Zelda Heardle Infinite is a customized version of the classic Zelda Heardle, designed to be played without a daily limit.
Each round selects a random song from The Legend of Zelda series, allowing you to play as long as you want.

This project was created for fun, with no commercial intent, and as a personal modification of existing projects.

---

## How to Play

1. Listen to the audio clip.
2. Type the name of the song or the game it belongs to.
3. Each attempt reveals hints:
   - Release year
   - Console
   - Game
4. You can skip attempts if you're unsure.
5. The goal is to guess the song in as few attempts as possible.

---
